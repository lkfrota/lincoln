Desvio Irradiação Solar (GHI) no Paraná

A síntese de irradiância global horizontal (GHI) é referência para geração elétrica fotovoltaica. A proporção com períodos anteriores serve como comparativo no monitoramento de geradores em operação. Para períodos anteriores é sempre usado o respectivo no ciclo solar, ou seja, de Janeiro a Abril só é comparado com Janeiro a Abril de um ou múltiplos anos.

Sobre o Relatório

Este relatório é uma síntese dos dados de Global Horizontal Irradiance (GHI) e Geração Distribuida Fotovoltaica (GD-FV) para o estado do Paraná. Visa oferecer uma visão resumida das condições recentes para esse setor produtivo. Os dados de irradiância foram obtidos da plataforma Oikolab, que disponibiliza o modelo climático ERA5, a mais recente geração de reanálise de dados produzido pela Copernicus Climate Change Service (C3S), implementado pelo ECMWF.

O GHI é um fator crítico para a estimativa de produção de energia, e esta análise visa fornecer informações de referência para engenheiros, projetistas, operadores e investidores, ajudando na tomada de decisões e na orientação de avaliação de performance de projetos de geração distribuída em operação.
